IsItNotGIF

A Laravel App that checks the URL you submit if it's animated or not. Intentionally made for streamers as a tool to check links made to deceive the user on the endpoint. The code requests the body and checks it for strings matching gif and mp4. Then it checks headers for MIME-type matching image/gif or video/mp4, finally it checks if the image is spoofed as an APNG (checks bytescode for specific byte-sequence). Also has a gitlab CI/CD pipeline that consists of build, testing and deployment.

ChestGrantedLOL

A Laravel App that displays what champions you have chest available on left this season. The app has a form where you fill in a league username and pick your designated server. Then it will fetch the required information by using Riot's Summoners-V4 API, Champion-Mastery-V4 API and Data Dragon API.
